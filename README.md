# Smart Home Automation System (C++)

## Description
This project simulates a smart home using OOP concepts in C++.

## Features
- Smart devices (Light, Fan, Camera)
- Uses inheritance and polymorphism
- Device control using base class pointers

## Concepts Used
- OOP (Encapsulation, Inheritance, Polymorphism)
- Abstract Classes
- Virtual Functions

## How to Run
g++ main.cpp -o app
./app
